# RepairMesh

RepairMesh est un agent de réparation autonome d'ordinateur assisté par IA.

## Documentation
- Architecture globale: `docs/00-ARCHITECTURE.md`
- Contrat IPC: `docs/01-IPC-CONTRACT.md`
- API gRPC/REST: `docs/02-API-GRPC-REST.md`
- Structure du projet: `docs/10-PROJECT-STRUCTURE.md`
- Workflow de collaboration IA: `docs/11-AGENT-WORKFLOW.md`
- Suivi d'avancement: `docs/12-PROGRESS-TRACKER.md`

## Organisation du dépôt
Le dépôt est préparé pour un développement collaboratif multi-agents via:
- une arborescence claire (`apps/`, `services/`, `packages/`, `infra/`, `scripts/`, `tests/`),
- des conventions de contribution (`AGENTS.md`),
- des templates de handoff (`docs/templates/`).
